# Certificate Generator

## Tech Stack
- Vite + React + TypeScript
- TailwindCSS
- html2canvas & jsPDF
- Deployment: Netlify / Vercel

## Installation
1. Clone the repo
   ```bash
   git clone <repo-url>
   ```
2. Install dependencies
   ```bash
   yarn install
   ```

## Development
```bash
yarn dev
```

## Build
```bash
yarn build
```

## Deployment
- **Netlify**: `yarn deploy`
- **Vercel**: Push to GitHub connected Vercel project
